# Spec: 每年都有收益吗？——分段检验

> 所有命令在沙箱外运行。

## 上下文

在 `q5-how-to-validate.ipynb` 中已有：
- 四个策略 vs 买入持有基准的整体指标对比
- 学员已了解基准、超额收益（Alpha）、卡玛比/索提诺比等概念

当前问题：整体指标是多年平均值，可能掩盖某些年份的亏损。需要逐年拆开看。

## 任务

在 notebook 中新建代码单元格，按年度拆解策略表现，并深挖 TopNRanking 在特定年份的异常。

## 要求

1. 阅读以下 oxq 接口的源码：
   - `result.daily_returns()` — 返回日收益率 Series
   - `result.equity_curve` — 净值序列（list of tuples: (date_str, value)）

2. 按年度拆解四个策略 + 基准：
   - 用 `daily_returns()` 按年分组
   - 计算每年的：年收益率（累乘）、年波动率、年夏普比、年内最大回撤
   - 同时计算等权买入持有基准的逐年收益

3. 打印逐年对比表：
   - 行：年份
   - 列：四个策略各一列（年收益），最后一列是基准年收益
   - 负收益用 `*` 标注

4. 画逐年收益柱状图（figsize 14x6）：
   - 每年一组柱子，每个策略一种颜色 + 基准灰色
   - y=0 处画一条水平线
   - 标题「逐年收益对比」

5. 深挖 TopNRanking 在负收益年份：
   - 用不同动量窗口 `mom_period = [5, 10, 15, 20, 25, 30]` 分别构造 `TopNRankingOptimizer(score_col="ram", n=3, filter_negative=True)` 并调用 `run_strategy(portfolio, indicators=[...])` 运行
   - 提取负收益年份的年收益率
   - 打印对比表：动量窗口 vs 该年收益
   - 如果所有窗口都亏钱，说明不是参数问题

6. 打印分析（根据实际数据动态描述方向）：
   - 哪些策略"每年都赚"？哪些有负收益年份？
   - 负收益年份中，基准表现如何？
   - TopNRanking 负收益年份深挖结论
   - 「平均年化 10% 听着不错——但如果其中一年亏了 20%，你扛得住吗？整体指标可以掩盖很多问题，逐年拆开才能看到真实的持有体验。」
   - 「TopNRanking 的负收益不是参数选错了——换遍所有窗口都亏。这是动量策略的结构性弱点：当市场反复上下波动、没有明确方向时，过去涨得好的不代表未来继续涨。动量指标有滞后性。」
   - 「知道了每年的表现——但每个月呢？赚钱是稳定地赚，还是靠几次大赚？」

## 结果呈现

1. 逐年收益对比表
2. 逐年收益柱状图
3. TopNRanking 异常年份深挖表
4. 分析文字
