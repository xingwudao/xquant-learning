# Spec: 和基准对比

> 所有命令在沙箱外运行。

## 上下文

在 `q1-strategy.ipynb` 中已有：
- 沪深300ETF 日线数据（变量 `df`，含 Close 列）
- 简单定投回测结果（变量 `result_dca`，含 return 列）

## 任务

计算「一次性买入持有」作为基准，和定投策略对比，帮助理解收益来源。

## 要求

1. 在 notebook 中新建代码单元格
2. 基准策略（Buy & Hold）：在数据起始日用全部资金一次性买入，持有到最后一天
3. 基准收益率 = (终点收盘价 - 起点收盘价) / 起点收盘价，存储在变量 `benchmark_return` 中
4. 计算基准的逐月累计收益率曲线（每月末收盘价相对起始价的涨跌幅），用于可视化对比
5. 与定投策略的最终收益率对比

## 结果呈现

1. 打印：沪深300ETF 起点价格、终点价格
2. 打印：一次性买入收益率、定投收益率、两者差异
3. 根据对比结果打印一句话说明：
   - 如果定投跑赢：「在下跌市中，定投通过分批买入避开了部分高位」
   - 如果定投跑输：「在上涨市中，越早全部投入越好」
4. 无论哪种结果，都补充一句：「但两种策略都没有 Alpha——收益完全取决于市场涨跌，这就是 Beta」
5. 画一张对比图（figsize 12×6）：定投累计收益率曲线 vs 一次性买入累计收益率曲线，图例标注两种策略名称
